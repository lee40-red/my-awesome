# My Awesome Project

אוסף של הדגמות אינטראקטיביות ופרויקטים ויזואליים

## קבצים בפרויקט

- `rag-tag-cag-interactive.html` - הדגמה אינטראקטיבית של טכניקות RAG, TAG, CAG
- `rag-tag-cag-explained.html` - הסבר מפורט על טכניקות RAG, TAG, CAG
- `machine-learning-visualization.html` - ויזואליזציה של למידת מכונה
- `protein-folding.html` - הדגמה של קיפול חלבונים
- `stocks-visualization.html` - ויזואליזציה של מניות
- `git-github-explained.html` - הסבר על Git ו-GitHub
- `index.html` - דף הבית

## שימוש

פתח כל קובץ HTML בדפדפן כדי לראות את ההדגמות האינטראקטיביות.

## טכנולוגיות

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Heebo, Space Mono)
